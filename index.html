About

Hello! I'm Hayden, an aspiring software developer in the Raleigh-Durham area. I was introduced to programming four years ago for a research project. Shortly after, I took my first Java programming class and I almost haven't missed a day since. For the next few years I hope to gain industry experience in game development and software engineering before I return to graduate school. I'm interested in learning more about graphics in game engines, simulation techniques, and artificial neural networks.

I have been at North Carolina State University for the past four years, while I have majored in Computer Science, and minored in Materials Science and Engineering. I have taken a variety of courses including: Software Engineering, Data Structures, Network Security, Foundations of Game Engines, Building Game A.I., Introduction to Graphics, Linear Algebra, Statistics, Multivariate Calculus, and Differential Equations.
